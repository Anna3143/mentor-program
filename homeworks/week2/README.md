## hw1：切版

請切出如下圖的版面：
![](example.png)

不需要做到完全一樣，但可以盡你所能做的相似，除此之外，還要支援 RWD，在手機上看也不會跑版。

你可以參考：[這裡](https://getbootstrap.com/docs/4.0/examples/jumbotron/)

考量到難度，以下幾點可以不用實作：

1. 上方的 Dropdown
2. 右上角的搜尋框跟按鈕
3. 手機版的 Navbar 可以不用顯示
